# CreditCardNumber

**CreditCardNumber** is built on top of [Material-UI's OutlinedInput](https://material-ui.com/api/outlined-input/) and provides an
`<input>` element that will format the 4 groups of digits of a credit-card number automatically for you.

## Default

```jsx
// live
<CreditCardNumber inputProps={{ autoFocus: true }} id="card-number" onChange={() => console.log('onChange')} />
```

## Controlled value validation

```jsx
// live
() => {
  const [value, setValue] = React.useState('')
  const [isError, setError] = React.useState(false)

  React.useEffect(() => {
    setError(!isValueValid(value))
  }, [setError, value])

  const isValueValid = value => {
    const trimedValue = value.replace(/\s/g, '')
    const regex = RegExp(/[\D]+/)
    return trimedValue === '' || !regex.test(trimedValue)
  }

  return <CreditCardNumber error={isError} id="card-number" onChange={e => setValue(e.target.value)} />
}
```

## Disabled

```jsx

```
