# Paragraph

**Paragraph** is built on top of [Material-UI's Typography](https://material-ui.com/components/typography/) and simply
extends it with a default `width` and `variant`. It was built to provide an optimal reading experience for text-contents
like in a blog.

## Default

```jsx
// live
<Paragraph>
  This is an awesome paragraph. It is based on the awesome material-ui Typography component. Its
  default variant is <code>body1</code>, and it has a maximal width of <b>650px</b>. According to awesome studies,
  it is the range where the eye feels the most comfortable reading. In addition to its variant,
  you can of course use all the{' '}
  <a href="https://material-ui.com/api/typography/" target="_blank" rel="noreferrer noopener">
    material-ui Typography component's props
  </a>
  .
</Paragraph>
```
