# Newsletter

**Newsletter** provides you with a built-in form to handle your emailing list, with a light interface to add text content
around.

- [Newsletter](/components/newsletter#)
- [Newsletter.Input](/components/newsletter#newsletter-input)
- [Newsletter.Button](/components/newsletter#newsletter-button)

## Default

```jsx
// live
() => {
  const [email, setEmail] = React.useState('')

  return (
    <Newsletter
      title='Subscribe to our newsletter'
      description='Get exclusive offers every week!'
      caption="By subscribing, you agree to receive emails from us. Don't worry, we are not spamers :)"
    >
      <Newsletter.Input id="email" placeholder="Email..." value={email} onChange={e => setEmail(e.target.value)} />
      <Newsletter.Button onClick={() => console.log('Thanks for subscribing!')}>
        Subscribe
      </Newsletter.Button>
    </Newsletter>
  )
}
```

## Newsletter.Input

### Default Input

```jsx
// live
() => {
  const [email, setEmail] = React.useState('')

  return <Newsletter.Input id="email" placeholder="Email..." value={email} onChange={e => setEmail(e.target.value)} />
}
```

### Various props

```jsx
// live
() => {
  const [email, setEmail] = React.useState('')
  return (
    <Newsletter.Input
      endAdornment={<span>:)</span>}
      id="email"
      color="secondary"
      placeholder="components-extra@gmail.com"
      value={email}
      onChange={e => setEmail(e.target.value)}
      label="Email"
    />
  )
}
```
